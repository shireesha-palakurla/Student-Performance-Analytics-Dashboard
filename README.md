## Overview

This project analyzes student academic performance using **Excel, MySQL, SQL, and Power BI**.
The dashboard helps visualize student insights such as pass/fail trends, grade distribution, subject-wise performance, and top-performing students.

---

## Objectives

* Analyze student performance data
* Create interactive dashboards
* Generate meaningful educational insights
* Demonstrate end-to-end data analytics workflow

---

## Tools & Technologies Used

* Microsoft Excel
* MySQL
* SQL
* Power BI

---

## Project Workflow

Dataset → Excel Cleaning → MySQL Database → SQL Analysis → Power BI Dashboard

---

## Dataset

Dataset used:
Student Performance in Exams Dataset from Kaggle

Main columns:

* Gender
* Math Score
* Reading Score
* Writing Score
* Lunch
* Test Preparation Course
* Parental Education

---

## Data Cleaning in Excel

Performed:

* Duplicate removal
* Column renaming
* Missing value checking
* Created calculated columns:

  * total_marks
  * average_marks
  * result
  * grade

---

## SQL Analysis Performed

SQL queries used for:

* Total Students
* Average Marks
* Top 10 Students
* Pass vs Fail Analysis
* Grade Distribution
* Gender-wise Performance
* Subject-wise Average Scores

---

## Power BI Dashboard Features

### KPI Cards

* Total Students
* Average Marks
* Passed Students

### Charts Used

* Pass vs Fail Pie Chart
* Grade Distribution Bar Chart
* Gender-wise Performance Column Chart
* Subject-wise Average Scores Chart

### Table Visual

* Top Performing Students

### Slicers

* Gender Filter
* Grade Filter
* Result Filter

---

## Insights Generated

* Most students achieved Grade B
* Reading scores were higher than math scores
* Female students performed better in reading
* Students completing test preparation performed better overall
